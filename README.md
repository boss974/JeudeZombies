# Jeu de Zombies

Projet de jeu zombie prévu en deux versions :

1. **Version Roblox** : expérience Roblox multijoueur avec lobby, téléporteur, arène, vagues de zombies, progression, boutique et classement.
2. **Version en ligne** : jeu web jouable depuis navigateur, avec mode solo puis mode multijoueur.

## Objectif

Créer un jeu simple à comprendre, nerveux, évolutif et monétisable proprement.

Le joueur commence dans un lobby, rejoint une zone de combat, survit à des vagues de zombies, gagne des récompenses et améliore sa progression.

## Versions prévues

### Roblox

- Lobby
- Téléporteur
- Arène
- Zombies par vagues
- Score
- Coins
- Boutique
- Classement
- Game Pass plus tard

### Web / Online

- Mode solo local
- Mode multijoueur en ligne plus tard
- Interface web simple
- Progression joueur
- Classement
- Architecture serveur sécurisée

## Priorité

1. Architecture propre
2. Prototype Roblox jouable
3. Prototype web solo
4. Système de vagues commun
5. Progression et sauvegarde
6. Multijoueur web
7. Monétisation Roblox
8. Version publique

## Méthode

Le projet doit rester clair pour être repris par plusieurs IA ou développeurs : ChatGPT, Claude, Codex, Roblox Studio et GitHub.

Chaque fichier doit être commenté simplement, avec une logique facile à relire.
