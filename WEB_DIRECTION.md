# Direction officielle du projet

Le projet ne doit plus être orienté Roblox.

## Direction retenue

Jeu web / navigateur en priorité.

## Version 1

- Mode solo
- Tower defense zombie
- Carte 2D simple
- Chemin sinueux
- Placement d'unités
- Vagues
- Boss
- Barre HP zombie optionnelle
- Tutoriel au premier lancement
- Ambiance calme dans le menu/lobby

## Version 2

- Multijoueur en ligne
- Comptes joueurs
- Sauvegarde progression
- Classements
- Saisons / événements

## Règle technique

Commencer sans dépendance lourde.

La première version doit pouvoir se lancer dans un navigateur avec HTML, CSS et JavaScript.

Ensuite seulement :

- serveur Node.js
- WebSocket
- base de données
- authentification

## Public visé

Jeune public + adultes.

Le jeu doit rester :

- non gore
- sécurisé
- simple
- coloré
- modéré
- sans chat libre au départ
