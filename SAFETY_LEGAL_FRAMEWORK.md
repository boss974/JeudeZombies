# Cadre sécurité et juridique

## Public visé

Le jeu vise principalement un public jeune, tout en restant accessible aux adultes.

La sécurité doit donc être prévue dès la conception.

## Principes

1. Protéger les mineurs.
2. Limiter les interactions toxiques.
3. Empêcher le harcèlement.
4. Éviter les contenus choquants.
5. Garder une modération simple.
6. Respecter les règles Roblox.
7. Respecter la protection des données.

## Sécurité dans le gameplay

Le jeu doit éviter :

- gore
- violence réaliste
- insultes affichées
- provocation entre joueurs
- système qui encourage l'humiliation
- classement humiliant
- noms d'équipes insultants
- messages personnalisés non filtrés

## Chat et communication

Recommandation forte :

- utiliser le chat Roblox filtré
- ne pas créer de chat maison non filtré
- limiter les messages rapides à des phrases prédéfinies
- pas de message privé interne au jeu au départ
- pas de système de clan libre au début

Messages rapides autorisés :

- Besoin d'aide
- Défends cette zone
- Améliore tes unités
- Boss en approche
- Bien joué
- Merci

## Anti-harcèlement

Prévoir :

- bouton signaler
- bouton mute joueur
- blocage des interactions abusives
- cooldown sur les pings
- limitation des emotes répétées
- pas de vote kick au début si cela peut être abusé
- logs serveur des actions importantes

## Modération contenu joueur

Au départ, éviter tout contenu créé librement par les joueurs :

- pas de nom de base personnalisable
- pas de texte libre sur panneaux
- pas d'image importée par joueur
- pas de description libre d'équipe

## Données personnelles

Ne pas collecter d'informations personnelles.

À ne jamais demander :

- vrai nom
- adresse
- téléphone
- email
- âge exact
- école
- réseaux sociaux

Données acceptables côté jeu :

- UserId Roblox
- score
- coins
- progression
- unités débloquées
- réglages de jeu

## Achats et monétisation

Règles internes :

- ne pas forcer l'achat
- ne pas promettre des gains trompeurs
- ne pas rendre le jeu impossible sans payer
- afficher clairement ce que le joueur achète
- privilégier les cosmétiques
- limiter la pression sur les mineurs

## Violence et ambiance

Style recommandé : cartoon apocalypse.

À éviter :

- sang réaliste
- démembrement
- cris réalistes excessifs
- horreur psychologique forte
- images choquantes

## Juridique - base de cadrage

Ce document n'est pas un avis juridique.

Pour cadrer correctement :

- respecter les règles Roblox Community Standards
- respecter les règles de maturité d'expérience Roblox
- respecter les règles de confidentialité Roblox
- ne pas collecter de données personnelles hors Roblox
- prévoir une page de règles du jeu
- prévoir une procédure de signalement
- conserver une logique de modération

## Page de règles en jeu

Texte simple à afficher :

Bienvenue.

Ce jeu doit rester respectueux.

Interdit : insultes, harcèlement, menaces, provocation, contournement des règles, exploitation de bugs, comportements toxiques.

Tu peux signaler ou bloquer un joueur si nécessaire.

Le but est de jouer en équipe et de s'amuser.

## Priorités de développement sécurité

1. Ne pas créer de chat maison.
2. Utiliser les outils Roblox.
3. Ajouter mute joueur.
4. Ajouter report joueur.
5. Limiter les textes libres.
6. Journaliser les actions serveur importantes.
7. Tester le jeu avec un compte enfant / ado.
